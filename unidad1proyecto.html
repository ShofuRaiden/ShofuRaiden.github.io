Unidad 1: Arquitectura de Computo

1.1 Módulos de Arquitectura

1.1.1 Clásicas

Las arquitecturas clásicas se basan en el modelo Von Neumann, donde la CPU ejecuta instrucciones almacenadas en memoria secuencialmente.

1.1.2 Segmentadas

La arquitectura segmentada divide la ejecución en etapas, mejorando la eficiencia al permitir la ejecución simultánea de distintas fases.

1.1.3 Multiprocesamiento

El multiprocesamiento utiliza múltiples procesadores trabajando en paralelo para mejorar el rendimiento y la capacidad de procesamiento.

1.2 Análisis de componentes

1.2.1 Arquitecturas

Se analizan los diferentes componentes estructurales que componen una arquitectura computacional.

1.2.1.1 Unidad Central de Procesamiento

Es el cerebro del sistema, encargado de ejecutar instrucciones y controlar el funcionamiento general.

1.2.1.2 Unidad Aritmética Lógica

Realiza operaciones matemáticas y lógicas básicas que son fundamentales para el procesamiento.

1.2.1.3 Registros

Pequeñas áreas de almacenamiento de alta velocidad dentro del procesador para datos temporales.

1.2.1.4 Buses

Canales de comunicación que transportan datos entre los diferentes componentes del sistema.

1.2.2 Memoria

1.2.2.1 Conceptos básicos del manejo de la memoria

Estudia cómo se organiza y gestiona la memoria para almacenar datos e instrucciones.

1.2.2.2 Memoria principal

Memoria RAM donde se almacenan los datos y programas en ejecución.

1.2.2.3 Memoria caché

Memoria rápida que almacena temporalmente datos de uso frecuente para acelerar el acceso.

1.2.3 Manejo de entrada/salida

1.2.3.1 Módulos de entrada/salida

Dispositivos y circuitos que gestionan la comunicación entre el sistema y el exterior.

1.2.3.2 Entrada/salida programada

La CPU controla directamente las operaciones de E/S mediante instrucciones específicas.

1.2.3.3 Entrada/salida mediante interrupciones

Permite que los dispositivos notifiquen a la CPU cuando requieren atención, optimizando recursos.

1.2.3.4 Acceso directo a memoria

El dispositivo de E/S puede acceder directamente a la memoria sin intervención de la CPU.

1.2.3.5 Canales y procesadores de entrada/salida

Mecanismos especializados para manejar múltiples operaciones de E/S en paralelo.

1.2.4 Buses

1.2.4.1 Tipos de buses

Se clasifican según su función: datos, direcciones y control.

1.2.4.2 Estructura de buses

Describe la organización física y lógica que permite la comunicación eficiente.

1.2.4.3 Jerarquía de buses

Relación entre buses principales y secundarios para optimizar el tráfico de información.

1.2.5 Interrupciones

Eventos que detienen temporalmente la ejecución normal para atender sucesos prioritarios.
